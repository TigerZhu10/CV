Projects

Arduino Alarm Clock

The Arduino Alarm Clock is a beginner-friendly project that helped me learn arrays, for loops, millis(), state machines, and circuit connections. It features real-time display, alarm setting, and a buzzer that sounds at the set time. The project required components like an Arduino Uno, 7-segment display, buzzer, push button, resistors, and jumper wires. Through this project, I improved my Arduino programming and circuit design skills. It was a great hands-on experience in both coding and electronics.
