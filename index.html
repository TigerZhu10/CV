Arduino Alarm Clock

The Arduino Alarm Clock is a beginner-friendly project that teaches programming and circuit design. It covers arrays, loops, millis(), state machines, and breadboard assembly. The clock displays real-time, allows alarm setting, and triggers a buzzer. It offers a hands-on learning experience in coding and electronics.

  • Push button debouncing for reliable input detection
  • Driving the WS5641AS 4x7 segment module without using external libraries
  • Designing a state machine to switch between different modes (real-time display, alarm setting, and buzzer activation)
  • Implementing an active buzzer for alarm notifications
  • Hardware circuit build and test on Breadboard
  • Using millis() as a software timer instead of relying on an external hardware timer

Pygame: Feed the dragon

"Feed the Dragon!" is a fast-paced arcade game where you control a dragon trying to catch flying coins. The dragon moves up and down using the arrow keys, avoiding missed coins that reduce lives. Each collected coin increases the player's score and gradually speeds up the game, making it more challenging. If all lives are lost, the game pauses, allowing the player to reset and try again.

  • Event Handling - Detects discrete and continues key presses and quits events for player control.
  • Collision Detection - Checks if the dragon catches the coin to update the score.
  • Game Loop - Continuously updates and redraws game elements smoothly.
  • Edge Detection - The dragon is restricted from moving beyond the top and bottom edges.
  • Coin Reset - The code resets the coin to the right with a random height if it moves off the left edge.
