Personal Website

This website was built using fundamental web technologies, ensuring a structured, responsive, and maintainable design. It includes essential HTML and CSS features, advanced layout techniques, version control, and Python integration for content management.

  • HTML Basics: Lists, links, images, divs, spans, paragraphs, favicon, and footer.
  • CSS Styling: Applied colors, fonts, box model, and positioning (static, relative, absolute, fixed, sticky).
  • Layout Techniques: Used Flexbox and Grid for advanced positioning and responsiveness.
  • Version Control: Managed code with Git and hosted on GitHub, utilizing branches
  • Dynamic Content & Responsiveness: Integrated Python scripts for modular content and adapted styles for different screen sizes.
